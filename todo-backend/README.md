## Description
This is the backend RestAPI for the todos

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## API URL
```
http://localhost:3000
```

## Endpoints

```
POST /todos - # Create a todo
GET /todos - # Get todo list
PUT /todos/items/:id - # Update a todo by id
DELETE /todos/items/:id - # Delete a todo by id
```
